# DigitalOcean Spaces Prefixes

Use stable top-level prefixes:

- `knowledge-base/`
- `instructions/`
- `workflows/`
- `tool-configs/`
- `evals/`
- `backlog/`
- `source-artifacts/`
- `run-summaries/`

Expected artifact examples:

```text
knowledge-base/YYYY/MM/DD/<run-id>/final-report.md
source-artifacts/YYYY/MM/DD/<run-id>/<source-id>.json
source-artifacts/YYYY/MM/DD/<run-id>/<youtube-video-id>-transcript.json
run-summaries/YYYY/MM/DD/<run-id>/summary.json
run-summaries/YYYY/MM/DD/<run-id>/trust-report.json
workflows/versions/research-workflow-v1.json
instructions/base.md
instructions/source-policy.md
instructions/notion-formatting.md
instructions/learning-output.md
tool-configs/openai-web-search.json
tool-configs/youtube.json
tool-configs/persistence.json
evals/cases/research-quality-v1.json
evals/results/YYYY/MM/DD/<run-id>/<eval-result-id>.json
```
